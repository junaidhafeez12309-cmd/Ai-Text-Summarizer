Terms and Conditions

These terms explain how to use the AI Text Summarizer responsibly and what we expect from everyone using the service.

One account per user

Users must not create multiple accounts. If we detect duplicate or false accounts, we can restrict access or remove those accounts.

Allowed Use

No spam or automation

Do not use automated tools or scripts to send requests. Avoid spam, repeated requests, or actions that may overload the service.

Responsible Use

No illegal or harmful activity

Do not misuse the tool for illegal, harmful, or unethical purposes. The service is meant for helpful, constructive use only.

Fair Usage

Limit excessive use

Use the service fairly and avoid excessive requests. If we see heavy use that impacts performance for others, we may limit or pause access.

Service Rights

We can block misuse

The service can restrict or block users if misuse is detected. This includes behavior that violates these terms or harms the platform.
